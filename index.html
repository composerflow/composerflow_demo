ComposerFlow

Abstract

ComposerFlow is a controllable music generation framework that aligns global style prompts and optional lyric guidance to produce coherent multi-model audio outputs. This demo showcases side-by-side generations from five systems given the same title, global style prompt, and lyrics, enabling qualitative comparison by ear.

This table lists demo tracks. Each row provides a title, a global style prompt, optional expandable lyrics, and five audio renderings produced by different models for quick A/B listening.
